# knowledge/

Project knowledge base — distilled from closed streams and accumulated over time.

- `.jsonl` files are git-tracked and append-only; one entry per distilled insight
- Run `agentboard distill <slug>` when closing a stream to extract knowledge
- Run `agentboard index` to build the local search index from all `.jsonl` files
- Run `agentboard search <query>` to find relevant knowledge before starting work
